# Online Bookstore System
(Spring Boot + Thymeleaf + SQLite)

## Prerequisites
Make sure you have these installed before starting:

- Java JDK 17

```
java -version
```

- VS Code

- VS Code Extensions

    - Extension Pack for Java

    - Spring Boot Extension Pack

- Git

```
git --version
```
## How to Set Up & Run
## 1️⃣ Clone the repository
(Do NOT download ZIP)

```
git clone https://github.com/YOUR_USERNAME/online-bookstore.git
```

Open the folder in VS Code.

## 2️⃣ Run the application
Option A (recommended):

- Open OnlineBookstoreApplication.java

- Click Run above main()

Option B (terminal):

```
mvnw spring-boot:run
```

## 3️⃣ Open in browser
```
http://localhost:8080
```

If the page loads, setup is successful ✅

## Database (SQLite)
- Database file: database.db

- Created automatically when the app runs

- Each developer has their own local database

❌ Do NOT push database.db to GitHub
